External UI skin folder (auto-loaded)
====================================

This project loads UI sprites from:
Assets/ArtDrop/FreeUI/

Expected filenames:
- ui_panel_bg.png
- ui_button_normal.png
- ui_button_highlight.png
- ui_button_pressed.png
- ui_button_disabled.png
- ui_icon_accept.png
- ui_icon_skip.png
- ui_icon_meta.png
- ui_icon_restart.png

No manual Unity hookup is required.
